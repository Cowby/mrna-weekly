## Late Evening Session: Deep Debugging & Unit Conversion (19:52-22:00)

### Critical Bugs Found & Fixed

#### 1. **Time Unit Conversion Bug** (commit 404a127) - CRITICAL!
**Problem:** Experimental data in HOURS, simulation expects SECONDS
- Dataset: `time: 168` (hours) → Code used 168 **seconds** (2.8 min) instead of 604,800s (7 days)!
- **Fix:** `duration = max_time_hours * 3600 * 1.1`
- **Impact:** Simulation now runs for correct duration (days instead of minutes)

#### 2. **Time Lookup Mismatch** (commit c92dd24) - CRITICAL!
**Problem:** Comparison looking up wrong simulation time points
- Experimental time in hours (6, 12, 24h) but comparison searched for 6, 12, 24 **seconds**
- All simulated values showed as 0.0 because t=6s had no protein yet
- **Fix:** Convert `t_exp_hours * 3600` before lookup
- **Impact:** Now finds correct simulation values at experimental time points

#### 3. **Unit Conversion System** (commits 192ae67, 194245c, d719a91, 7cc9f9b, a32c06b)
**Problem:** Simulation outputs Molarity, experiments measure ng/mL, pg/mL, RLU, fold-change
- Created `cell_twin/validation/unit_conversion.py` with priority-based conversion
- **Conversions implemented:**
  - EPO: M × 34,000 g/mol × 1e6 → ng/mL
  - Cytokines: signal (0-1) × max_concentration → pg/mL
    - TNF-α: 0-1 → 0-1000 pg/mL
    - IFN: 0-1 → 0-100 pg/mL
  - Luciferase: M × 1e9 × 1000 → RLU
  - Fold-change: (M / baseline_M) where baseline = 1e-7 M
  - Phosphorylation: 0-1 (no conversion)

**Priority order matters:**
1. Specific observables (EPO, cytokines with _pg_mL suffix)
2. Explicit units check (skip fold-change if has _pg_mL, _ng_mL, etc.)
3. Value-based heuristic (0.1-10 → fold-change)
4. Generic protein (ng/mL)
5. Default (no conversion)

#### 4. **Wrong YAML Files Loaded** (commit 4d3b0ae)
**Problem:** Validation script loaded old placeholder YAMLs instead of REAL paper-extracted data
- `kariko_2011_hplc.yaml` (old) had `TNF_alpha` without `_pg_mL` suffix
- `kariko_2011_hplc_REAL.yaml` (new) had correct `TNF_alpha_pg_mL`
- **Fix:** Renamed `_REAL.yaml` → `.yaml`, moved old to `_OLD.yaml`

#### 5. **Cytokine Mapping** (commit d30f60d)
- Fixed: `signal.tnf_alpha` → `signal.cytokine_tnfa` (correct column name)
- Fixed: `signal.il6` → `signal.cytokine_il6`
